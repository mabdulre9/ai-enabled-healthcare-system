# Documentation for GitHub Pages

Clean, professional documentation inspired by OpenCV style.

## Features

- Professional design (no emojis, clean colors)
- Multi-page structure with sidebar navigation  
- Custom layout resembling OpenCV documentation
- Dark blue/gray color scheme
- Responsive design

## Deployment

1. Upload `docs` folder to your repository root
2. Go to Settings → Pages
3. Select: main branch → /docs folder
4. Save

Documentation will be live at:
```
https://mabdulre9.github.io/ai-enabled-healthcare-system/
```

## Structure

- Custom Jekyll layout in `_layouts/default.html`
- Professional styling (no green theme)
- Sidebar navigation
- Clean header and footer
- Multi-page documentation

## Pages

- index.md - Homepage
- getting-started.md - Installation guide

- And more...

All pages use the professional custom layout.
