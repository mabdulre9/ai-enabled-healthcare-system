# 🚀 Ollama Setup Guide - EASIEST Local AI!

## Why Ollama? (vs llama-cpp-python)

✅ **No C++ compiler needed** - Just download & install  
✅ **No build tools required** - Works instantly  
✅ **Easy model management** - One command downloads  
✅ **Automatic updates** - Models stay current  
✅ **Background service** - Always ready  
✅ **Cross-platform** - Windows, Mac, Linux  

❌ llama-cpp-python requires Visual Studio Build Tools (5GB download!)  
✅ Ollama just works!

---

## 📥 Step 1: Install Ollama

### **Windows:**
1. Visit: https://ollama.com/download/windows
2. Download `OllamaSetup.exe`
3. Run installer
4. Ollama starts automatically!

### **Verify Installation:**
```cmd
ollama --version
```

Should show: `ollama version is 0.x.x`

---

## 🤖 Step 2: Download AI Model

```cmd
# Download Qwen 2.5 4B (Recommended - Fast & Smart)
ollama pull qwen2.5:4b
```

**This downloads ~2.7GB and takes 2-5 minutes.**

---

## ⚙️ Step 3: Configure Healthcare System

**1. Install Ollama Python Client:**
```cmd
pip install ollama
```

**2. Run System & Configure:**
- Open Settings → Select **Ollama mode**
- Model: qwen2.5:4b
- Save Settings

**3. Use AI Assistant!**
- Opens patient → AI Assistant
- Shows "OLLAMA" mode 🟢
- Completely offline!
